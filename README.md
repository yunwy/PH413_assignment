# PH413_assignment
Assignment repository for computational physics course of KAIST(PH413)

These codes were written in miniconda environment and following libraries are required:

<pre><code>
  conda install -c conda-forge python=3.11.14
  conda install -c conda-forge jupyter matplotlib scipy scikit-learn scikit-image h5py tqdm
  pip install numpt==1.26.4
  pip3 install torch torchvision
  pip install Pillow==10.1
</code></pre>
